PROMPT BUILDER - PORTABLE VERSION
==================================

This is a portable version of Prompt Builder that can run without installation.

REQUIREMENTS:
- Windows 10/11 (64-bit)
- At least 4GB RAM (8GB+ recommended for large models)
- GGUF model file (optional, for local LLM features)

QUICK START:
1. Double-click "start.bat" to launch the application
2. If you have a GGUF model, place it in the "models" folder
3. The application will open in a new window

MODEL SETUP:
- Place your GGUF model files in the "models" folder
- Recommended model: Qwen2.5-7B-Instruct-Q4_K_M.gguf
- The app will work without models but local LLM features will be unavailable

TROUBLESHOOTING:
- If the app doesn't start, try running "start.ps1" instead
- Make sure all files are extracted to the same folder
- Check that Windows Defender isn't blocking the executable
- Ensure you have sufficient disk space and RAM

FEATURES:
- AI-powered prompt generation
- Local LLM support (with GGUF models)
- Image generation prompt building
- Multiple preset categories
- Export/import functionality

For support or updates, visit: https://github.com/btitkin/promptbuilder
